Design Brief

Project Overview

Design and develop a centrifugal pump system for automotive paint shop coolant circulation at Tesla's Fremont facility. The pump must handle corrosive coolant mixtures at elevated temperatures while meeting strict uptime requirements and API 610 standards.

Performance Requirements

  • •Flow Rate: 500 GPM (31.5 L/s)
  • •Discharge Pressure: 150 PSI (10.3 bar)
  • •Fluid: Corrosive coolant, pH 8-10
  • •Temperature: 180°F (82°C) continuous

Design Constraints

  • ✓Standards: API 610 BB1, API 682 Plan 11
  • ✓Uptime Target: 99.5% availability
  • ✓Maintenance: Back-pullout design required

Engineering Decisions

1. Configuration: API 610 BB1 Back-Pullout Design

Selected BB1 (bearing frame, single-stage, overhung impeller) configuration to enable complete rotor assembly removal without disturbing piping. Critical for minimizing downtime in production environment.

2. Material Selection: 316 Stainless Steel

Casing
316 SS Cast
Impeller
316 SS Cast
Shaft
17-4PH SS

316 stainless steel selected for all wetted components to resist chloride-induced stress corrosion cracking. 17-4PH for shaft achieves required yield strength (170 ksi).
